My website project: Twist Your Noodles

Project Overview

  • Academic web development project
  • Brand Concept: Crochet, Identity and Slow Fashion
  • Focus: Storytelling, Aesthetics and User experience

The Idea

  • Official web presence for my artisan brand
  • Promoting Slow Fashion, educating the customer on the importance of handmade goods, and contrasting disposable Fast Fashion practices.
  • The site tells a story rather than only selling

Website Structure

  • Home: Hero, brand introduction
  • About Us: Values and inspiration
  • The Lab: Production steps and process of realization
  • Products: Carousel with highlightes pieces
  • Contact us: Functional contact form

Technologies Used

  • HTML: semantic structure
  • CSS: variables, animations and responsive styling
  • Bootstrap5: grid, components, responsive utilities
  • JavaScript: form validation, cookie banner
  • Google Fonts and Font Awesome: typography and icons

Navigation Structure

  • The Navbar uses standard Bootstrap classes.
  • Key Links (Section Anchors):
    • #home, #about, #lab, #products.
    • External link to contacts.html and Instagram.

Section Highlights: About Us & The Lab

The ‘About Us’ Story

  • The section focuses on the brand’s origin: the passion for crochet
  • Establish an official online presence that functions not just as an e-commerce platform, but as a narrative vehicle for the brand’s unique identity and philosophy.

The Lab: The Noodle Journey

  • This section details the slow-fashion process using a clear 3-step grid layout.

  • Code Implementation:

    • Icons: Using Font Awesome for visual clarity.


Product Carousel

  • Implementation: Using the Bootstrap Carousel component.
  • Functionality: Allows interactive display of 5 key items (item1.jpg to item5.jpg).


Moving Strip

  • A simple, yet effective, design element for brand reinforcement.
  • Code Insight: This uses CSS keyframes (defined in style.css) to create an infinite horizontal scroll effect for the text: TWIST YOUR NOODLES •.

Footer and Compliance

  • Structure: Standard four-column layout within the footer.
    • Includes Quick Links, Contact Info, and Social Media links.
  • Compliance Feature: The inclusion of a fixed, styled Cookie Banner and a JavaScript handler (shows attention to modern web standards and user experience.)

Dedicated Contact Page:

  • This page fulfills the essential e-commerce requirement of clear customer communication.
  • It maintains the brand aesthetic using the same Bootstrap 5 and custom CSS variables.

Form Implementation

  • The form is structured using Bootstrap’s grid system for a clean layout.
  • Key Feature: External Submission Handling.
    • The form action uses a third-party service: action="https://formsubmit.co/balloilenia@gmail.com"
    • This allows email submissions without server-side scripting, making the project portable and easy to deploy.

Long-Term Plans

  • Integrate a real E-commerce system
  • Add payment gateways
  • Improve multilingual support

Lessons Learned

  • Structuring a multi-page site with reusable patterns
  • Effective use of Bootstrap components and utilities
  • Managing CSS variables for theming
  • Writing JavaScript (validation, localStorage)

Lessons Learned - Design

  • Importance of visual hierarchy
  • Balancing images with short, readable text
  • Choosing a consistent color palette and fonts
  • Testing responsiveness and micro-interactions

Final Conclusions

  • Twist Your Noodles is a creative prototype that merges design, craft and web development
  • The project is ready to transition into a full e-commerce platform.
  • It demonstrates both technical skill and brand storytelling
